# AspectDemo
hook demo 加  皮肤切换
之前公司做了皮肤切换，是同事做的，代码写得超级复杂，居然还有hook

仔细复盘后，发现就是个笑话，hook 代码一句都没有用。

实际就是获得颜色值，在界面加载的时候改变控件颜色，毛技术含量都没有。

居然被忽悠了这么久，完全是个马保国啊。

汗。。。

思路：

1：建个类管理皮肤，用字典的方式写好皮肤的各个模板；

2：控件设置颜色的时候，用是皮肤的模板颜色；
